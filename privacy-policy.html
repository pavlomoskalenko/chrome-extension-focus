Privacy Policy — Focus Mode

Last updated: 15 April 2026

Focus Mode is a browser extension for Google Chrome that blocks websites you add to a list while Focus Mode is enabled, and shows a simple timer for how long you stay in focus mode.

Data we collect

We do not operate backend servers for this extension. We do not collect, transmit to ourselves, sell, or use your browsing history, passwords, or personal identifiers through the extension.

Data stored on your device

The extension stores information only using Chrome’s extension storage APIs, for example:

  • Your list of sites to block
  • Whether Focus Mode is on or off
  • Values used for the session timer and total focus time

If you use Chrome while signed in and have Chrome sync enabled for extensions, Google may sync this data according to your Google account and Chrome settings. That processing is governed by Google’s policies, not by a separate service run by the extension developer.

How permissions are used

  • Storage — to save your settings and block list locally (and via Chrome sync when applicable).
  • Declarative Net Request — to block network requests to domains you listed, only while Focus Mode is on.
  • Optional broad host access — only if you approve it in Chrome’s prompt, so the extension can apply block rules to the domains you add. It is requested when you turn Focus Mode on (with sites in your list) or when you allow blocking from the popup. The extension does not use this access to read unrelated page content.

Remote code

The extension does not load or execute code from remote servers. All extension logic is contained in the packaged files distributed through the Chrome Web Store.

Changes

If this policy changes in a meaningful way, we will update this page and, where required, the Chrome Web Store listing.
